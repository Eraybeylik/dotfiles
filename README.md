# 🏠 xera's dotfiles

**Arch Linux · Hyprland · Catppuccin Mocha · Nvidia**

</div>

## 📸 Kurulum
```bash
# 1. Repoyu klonla
git clone git@github.com:Eraybeylik/dotfiles.git ~/.dotfiles

# 2. Scripti çalıştır
cd ~/.dotfiles
chmod +x install.sh
./install.sh

# 3. Reboot
sudo reboot
```

## 📦 İçerik

| Config | Açıklama |
|--------|----------|
| `hyprland` | Compositor, keybinds, hypridle, hyprlock |
| `waybar` | Status bar |
| `kitty` | Terminal emulator |
| `dunst` | Bildirim yöneticisi |
| `tofi` | Uygulama launcher |
| `wlogout` | Logout menu |
| `nvim` | Neovim (LazyVim) |
| `btop` | Sistem monitör |
| `starship` | Shell prompt |
| `zsh` | Shell + aliases |

## ⌨️ Keybindler

| Kısayol | Aksiyon |
|---------|---------|
| `SUPER + RETURN` | Terminal (kitty) |
| `SUPER + W` | Browser (Chrome) |
| `SUPER + R` | Launcher (tofi) |
| `SUPER + E` | Dosya yöneticisi |
| `SUPER + Q` | Pencere kapat |
| `SUPER + F` | Float toggle |
| `SUPER + SHIFT + F` | Fullscreen |
| `SUPER + L` | Ekran kilitle |
| `SUPER + ESC` | Logout menu |
| `SUPER + V` | Clipboard geçmişi |
| `SUPER + P` | Renk seçici |
| `Print` | Ekran görüntüsü |

## 🖥️ Donanım

- GPU: Nvidia (driver: nvidia-dkms)
- Çift monitör: HDMI-A-1 + eDP-1
